Script Configuration Details

Field Name Description & Examples
Script Argument List This field specifies arguments to run the Python script with. Arguments can be configured in two ways: 1. List the arguments in order, for example: '1,2,3 Android Nuwa'. To use them in the script, reference them as follows:
import sys
str = sys.argv[2] # this will give you 'Nuwa'
print str
 2. List the arguments with explicit names, for example: '--systemVersion=Oreo' and reference the name directly in the script as follows:
import argparse
parser = argparse.ArgumentParser(description='manual to this script')
parser.add_argument('--systemVersion', type=str, default = None)
args = parser.parse_args()
print args.systemVersion
Script Code Content This field contains the code content of the script. Note that you should import the 'android' library first, as this provides APIs to interact with the connected device. Refer to this link for more information on the available APIs. Please note that the script should conform to Python2 syntax. The following is an example Python script to trigger a notification on the screen:
import android
droid = android.Android()
droid.makeToast('Hello, Android!')
